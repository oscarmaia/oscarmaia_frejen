## Instalação

Para instalar as dependências do projeto, execute:

```bash
npm install
```

## Uso

Para iniciar o projeto, utilize o comando:

```bash
npm run dev
```

## Testes

Para executar os testes, utilize o comando:
```bash
npm test
```

Os testes são escritos utilizando o framework Jest.
